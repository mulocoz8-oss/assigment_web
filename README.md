This project is a web-based O-Level Student Management System designed according to the Tanzania Education System.

The system manages students from Form I to Form IV and allows teachers to:

- Register students
- Manage class-based dashboards
- Enter subject results
- Track academic performance
- Analyze subject performance
- Calculate student averages

The system is built using:

- HTML (Structure)
- CSS (Styling)
- JavaScript (Logic and Interaction)

All data is stored using JavaScript arrays and objects.
